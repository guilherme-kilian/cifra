Instruções de Execução:

Execute a main e responda às perguntas que aparecerão no console.

* O número de deslocamento define quantas posições no alfabeto cada letra será avançada.
Por exemplo: Se a letra for A e o deslocamento for 5, após a criptografia, ela se tornará um F.

O código não roda em loop. Isso significa que, após criptografar, você pode escolher descriptografar ou não, depois disso o programa para e será necessário executar o código novamente para realizar outro processo.
